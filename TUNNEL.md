# 内网穿透部署指南

## 一、cpolar 内网穿透（推荐）

### 1. 下载安装 cpolar
访问：https://www.cpolar.com/
- 注册账号（邮箱注册）
- 下载 Windows 客户端
- 安装并登录

### 2. 配置隧道

**方法A: 使用图形界面**
1. 打开 cpolar 客户端
2. 点击"隧道管理" → "创建隧道"
3. 配置参数：
   - **隧道名称**: act-record
   - **协议**: http
   - **本地地址**: 3000
   - **域名类型**: 随机域名（免费）或 自定义域名（付费）
4. 点击"创建"

**方法B: 使用命令行**
```bash
# 启动项目
npm run dev

# 新开一个终端，启动 cpolar
cpolar http 3000
```

### 3. 获取访问地址
- cpolar 会生成一个公网地址，例如：
  - `https://xxxxx.cpolar.cn`
- 分享这个地址给团队成员即可访问

### 4. 保持隧道稳定（推荐升级）
**免费版限制**:
- 每次重启后域名会变
- 带宽 1-2Mbps
- 需要每天重新获取地址

**付费版优势**（¥10/月起）:
- 固定域名（不会变）
- 更高带宽
- 更稳定

## 二、ngrok 内网穿透（国际）

### 1. 安装 ngrok
访问：https://ngrok.com/
- 注册账号
- 下载 Windows 版本
- 解压到任意目录

### 2. 配置认证
```bash
# 在 ngrok 目录下运行
ngrok config add-authtoken <你的token>
```

### 3. 启动隧道
```bash
# 启动项目
npm run dev

# 新开终端，启动 ngrok
ngrok http 3000
```

### 4. 访问地址
ngrok 会显示：
```
Forwarding  https://xxxx-xx-xx-xx-xx.ngrok-free.app -> http://localhost:3000
```

**注意**: ngrok 在国内访问较慢，推荐使用 cpolar

## 三、花生壳内网穿透（国内老牌）

### 1. 下载安装
访问：https://hsk.oray.com/
- 注册账号
- 下载客户端
- 安装并登录

### 2. 配置映射
1. 点击"内网映射" → "添加映射"
2. 配置：
   - **应用名称**: 校研会系统
   - **内网主机**: 127.0.0.1
   - **内网端口**: 3000
   - **外网域名**: 免费随机域名
3. 保存并启动

### 3. 访问
使用分配的外网域名访问

**限制**: 免费版每月流量限制 1GB

## 四、使用内网穿透的完整流程


## 五、常见问题

### Q1: 域名每天都变怎么办？
**A**: 升级为付费版（¥10-30/月），获得固定域名

### Q2: 电脑关机后别人还能访问吗？
**A**: 不能。必须保持电脑开机且项目运行

### Q3: 访问速度慢怎么办？
**A**:
- cpolar: 国内访问较快
- ngrok: 国外服务，国内较慢
- 升级付费版获得更高带宽

### Q4: 多个人同时访问会卡吗？
**A**:
- 免费版带宽有限，5-10人同时在线可能会卡
- 付费版或云服务器会更流畅

### Q5: 电脑性能影响大吗？
**A**:
- CPU占用：10-30%（取决于访问量）
- 内存占用：约500MB-1GB
- 建议配置：8GB内存以上

## 九、保持电脑长时间运行的设置

### Windows 电源设置
1. 控制面板 → 电源选项
2. 选择"高性能"模式
3. 更改计划设置：
   - 关闭显示器：30分钟
   - 使计算机进入睡眠状态：**从不**

### 防止自动更新重启
1. 设置 → Windows 更新
2. 暂停更新 7 天
3. 活动时段设置为全天

### 性能优化
- 关闭不必要的后台程序
- 确保散热良好（笔记本建议加散热垫）
- 定期清理垃圾文件
